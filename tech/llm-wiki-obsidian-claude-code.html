📖 정의

개인 지식 DB 위에서 AI가 직접 읽고 추론하는 시스템

  • • 단순 메모 저장 도구가 아님

🃏 제테 카르텐 (Zettelkasten)

독일 사회학자 루만이 개발한 메모 시스템. 아이디어 하나당 카드 하나, 카드끼리 링크로 연결. 단순 저장이 아닌 새 아이디어가 자연스럽게 생겨나는 구조.
