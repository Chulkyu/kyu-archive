🧰 Caveman 부가 기능 4가지

📝

① 부가 기능

Caveman Commit

Git 커밋 메시지를 원시인처럼 짧게 작성해줘요. 50글자 이내, 무엇을 했는지가 아니라 왜 했는지 위주로.

🔍

② 부가 기능

Caveman Review

PR 리뷰를 한 줄로 끊어줘요. 장황한 코드 리뷰에서 핵심만 뽑아내는 기능.

📊

③ 부가 기능

Caveman Stats

Caveman 덕분에 지금까지 얼마나 토큰을 절약했는지, 달러로 얼마나 아꼈는지 보여줘요. Claude Code 상태바에 게임 점수처럼 표시.

🗜️
